<a name="faq_latestfw" />**Q:** *I am on the latest system version, is my device hackable?*    
**A:** Yes, but not natively, we can, however, hack the PSP Emulator within the Vita (ePSP)
<br><br>
<a name="faq_updatecfw" />**Q:** *How do I update HENkaku?*    
**A:** Uninstall Ensō, run HENkaku, reinstall Ensō.
<br><br>
<a name="faq_need" />**Q:** *What do I need to do this guide?*    
**A:** This guide can be completely done with free software.
<br><br>
<a name="faq_risky" />**Q:** *How risky is hacking my console?*    
**A:** Bricks are relatively rare if you know what you are doing and don't modify critical system files.
<br><br>
<a name="faq_homebrew" />**Q:** *Can I run awesome homebrew and emulators with this?*    
**A:** Yes! HENkaku enables native Vita homebrew, and the ePSP exploits enable using PSP homebrew.
<br><br>
<a name="faq_updates" />**Q:** *Is it safe to update after getting HENkaku Ensō?*     
**A:** No. You will lose access to HENkaku. You can update to a modified version of 3.65 with this [HowTo](update-365)
<br><br>
